# 巨物清洁工 Unity 版

目标：微信小游戏 / Unity WebGL 转小游戏。第一关：真实感自行车清洁。

## 场景说明

启动场景由 `Assets/Scripts/GameBootstrap.cs` 运行时生成，不依赖手工摆场景：

- 第一人称摄像机
- 横屏 FPS 操作
- 真实比例自行车模型（由基础几何体组合生成）
- 轮胎、车架、车把、车座、链条/飞轮等部件
- 泥污/油污贴片
- 高压水枪、水柱、泡沫/水雾粒子
- 清洁进度、通关判定

## 操作

- 左侧虚拟摇杆移动
- 右侧拖动转向
- 按住喷水
- 准星对准自行车污渍进行清洗

## 构建目标

Unity WebGL 后接微信小游戏转换插件。
